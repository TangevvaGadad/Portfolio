## Technologies Behind PrepGen

- **Next.js**: Core framework for full-stack web development.
- **shadcn/ui**: Modern, reusable UI component library for clean and consistent design.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI styling.
- **Prisma & NeonDB**: Modern, serverless relational database with powerful ORM.
- **Clerk Authentication**: Secure, easy-to-use auth for user accounts.
- **Zod**: Robust schema validation for data integrity.
- **Inngest**: Serverless job queues and scheduling.
- **Gemini API (Google AI)**: AI model powering content generation tools.
- **React Hook Form & React Query**: For seamless form handling and data fetching.
- **Recharts**: Interactive and responsive data visualization for performance tracking.
- **Sentry**: Error monitoring and performance analytics.
